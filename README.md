# schoolvisits
As an English teacher in Hokkaido, I routinely visit three other schools, in addition to my main school. 
I’d like to design a program that will make it easier to manage tracking these trips. 
I would like the program to be able to: 1. Keep track of the visit dates and which school I will be visiting, 2. Tell me how many visits I have in the remaining school year, and 3. Tell me when the next visit is.
- object-oriented programming approach
